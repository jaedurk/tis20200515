Vue 컴포넌트 만드는 방법을 배운다. 컴포넌트는 재사용 가능한 코드 template을 보완하여 만든 요소다. Vue 컴포넌트는 인스턴스의 모든 옵션을 사용할 수 있습니다 component는 HTML템플릿과 더불어 데이터 셋팅, 함수 정의 등 다양한 작업을 할 수 있다. component는 Vue.js의 핵심 요소이다. 재사용이 가능하다. 여러개의 Component 사용이 가능하다. Vue.js 에서 제공하는 컴포넌트를 이용해 다양한 템플릿을 만들어 사용할 수 있다. Vue 컴포넌트 종류 전역 컴포넌트는 Vue.Component(tagName, options)로 정의되는 컴포넌트 tagName은 kebab-case를 사용한다. 소문자와 하이픈만 사용. 예시) my-component 전역 컴포넌트는 Vue.Component(tagName, options)를 사용하여 정의한다. Vue.component("my-component", { // 옵션 }); 사용는 커스텀 엘리먼트로 <my-component></my-component>로 사용할 수 있습니다 <div id="example"> <my-component><!-- 전역 컴포넌트 --></my-component> </div> // 전역 컴포넌트 등록 Vue.component("my-component", { template: "<div>사용자 정의 컴포넌트 입니다!</div>" }); // 루트 인스턴스 생성 new Vue({ el: "#example" }); 지역 컴포넌트는 Vue 인스턴스의 components 옵션에 정의되는 컴포넌트 var Child = { template: "<div>사용자 정의 컴포넌트 입니다!</div>" }; new Vue({ // ... components: { // <my-component> 는 상위 템플릿에서만 사용할 수 있습니다. "my-component": Child } }); 왜 전역/지역 컴포넌트를 사용하는가? 컴포넌트를 재사용하기 위해서. Vue 컴포넌트는 Vue 인스턴스의 모든 옵션을 사용할 수 있다. Vue 컴포넌트 생성시 주의 사항 Vue 컴포넌트 data 옵션은 함수. Vue 인스턴스 data 옵션은 객체. <div id="example-2"> <simple-counter></simple-counter> <simple-counter></simple-counter> <simple-counter></simple-counter> </div> Vue.component("simple-counter", { template: "<button v-on:click='counter += 1'>{{ counter }}</button>", // 데이터는 기술적으로 함수이므로 Vue는 따지지 않지만 // 각 컴포넌트 인스턴스에 대해 같은 객체 참조를 반환합니다. data: function () { return { counter: 0 } } }); new Vue({ el: "#example-2" }); [참조] 교재 page 52~61 https://kr.vuejs.org/v2/guide/components.html
