Vue에서의 Webpack 실습을 위해서는 Vue-CLI 를 사용해야 한다.

      # CLI 는 command line interface의 약자.

      step 1. mkdir webpack-test                    #폴더만들기
      step 2. cd webpack-test                       #폴더만들기
      step 3. npm install vue-cli                   #Vue-CLI 다운로드
      step 4. npx vue init webpack-simple           #Vue 프로젝트 생성(책에는 없음)
      step 5. npm install                           #Vue 프레임워크 다운르도
      step 6. npm run dev                           #Vue 프로젝트 실행
      step 7. 실습 파일 복사
        Header.vue, Footer.vue, App.vue 파일을 webpack-test/src/ 에 붙여넣기
      
      step 8. npm run build                         #webpack-test/dist/ 에 합쳔진 파일 만들기
      step 9. npm run dev                           #Vue 프로젝트 실행
