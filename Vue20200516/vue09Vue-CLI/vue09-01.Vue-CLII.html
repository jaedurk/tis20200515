Vue에서의 Webpack 실습을 위해서는 Vue-CLI 를 사용해야 한다.

      # CLI 는 command line interface의 약자.

      step 1. mkdir webpack-test
      step 2. cd webpack-test
      step 3. npm install vue-cli
      step 4. npx vue init webpack-simple
      step 5. npm install
      step 6. npm run dev
      step 7. 실습 파일 복사
        Header.vue, Footer.vue, App.vue 파일을 webpack-test/src/ 에 붙여넣기
