vue.js 에서의 style 속성 설정 방식을 학습한다. style 속성으로 설정할 값은 data 옵션에 저장. style 속성 설정은 v-bind:style 디렉티브를 사용하여 작성. single style binding: new Vue({ data : { baseStyles: { fontWeight:'800', color: 'red' }, } }); <p v-bind:style="baseStyles"> baseStyles and overrideStyles </p> multi style binding: new Vue({ data : { baseStyles: { fontWeight:'800', color: 'red' }, overrideStyles: { color:'blue' }, } }); <p v-bind:style="[baseStyles, overrideStyles]"> baseStyles and overrideStyles </p>
Vue.js 를 사용한 style single 바인딩 복습
